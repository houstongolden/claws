# Telegram channel mapping

Map Telegram chat IDs and topic IDs to Claws conversations.

Format:
- `telegram_topic:` — Telegram chat_id or forum topic id (number)
- `maps_to:` — Claws destination: `#channel-slug` or `project:project-slug`

---

telegram_topic: 12345
maps_to: #sales

telegram_topic: 55555
maps_to: project:claws-so
