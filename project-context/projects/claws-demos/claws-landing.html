ClawsShip in the shell

One workspace for vibe-coded artifacts, research with sources, and tasks that stick—without handing your context to a black box you don’t control.

What Claws does

01

Threads that survive

Sessions stay on your machine. Switch chats, close the tab, come back—same sidebar, same history, same thread IDs. Nothing evaporates because a server forgot you.

02

Build you can open

The agent writes real files—HTML, markdown, whatever the tree allows. Preview in the canvas, iterate in chat. Shipping means a path on disk, not a screenshot.

03

Tools, not theater

Search, fetch, memory, approvals—wired for agents that actually do work. Plan mode when you want read-only; agent mode when you’re ready to commit.
