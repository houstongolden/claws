## Pipeline Stages

| Stage | Definition | Goal |
|-------|-----------|------|
| **Lead** | Identified, not yet contacted | Qualify or discard within 5 days |
| **Qualified** | Confirmed ICP fit, problem validated | Book a demo |
| **Demo** | Demo scheduled or completed | Send proposal within 48h |
| **Proposal** | Proposal sent | Follow up every 3 days until decision |
| **Closing** | Verbal yes, working on contract | Close within 2 weeks |
| **Won** | Signed and paying | Handoff to onboarding |
| **Lost** | Closed lost | Document reason |

---

## Follow-up Rules (Agent enforces these)

- **Lead → no response in 5 days:** Agent drafts a follow-up for approval
- **Qualified → no demo booked in 7 days:** Agent flags as at-risk
- **Demo done → no proposal in 48h:** Agent flags and drafts proposal prompt
- **Proposal out → no response in 5 days:** Agent drafts follow-up
- **Any deal → no activity in 7 days:** Agent alerts as stale
